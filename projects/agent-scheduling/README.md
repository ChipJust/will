# Agent Scheduling — Cohort Coordination via Chat

A cohort of seven runs three triad meetings per month plus one full-cohort meeting. Coordinating seven calendars manually fragments the cohort. This project delivers a tool where each participant's personal agent negotiates meeting times with the others' agents, surfacing to the human only when input is needed.

Originally drafted as a problem note 2026-04-22; migrated into the projects template 2026-04-26.
